So what version of ruby should I use?

2.7.1 is the lastest release

You can use a version management tool to switch between ruby versions and set default versions

RVM

rbenv

chruby

RVM

  • Has more features than rbenv and chruby
  • Includes a built-in ruby installation tool
  • Heaviest overall

rbenv

chruby

RVM

rbenv

  • Has way fewer features than rvm
  • Keeps things simple and light weight
  • Considered to be more "developer friendly"
  • Can use plugins for additional functionality

chruby

RVM

rbenv

chruby

  • New kid on the block
  • Even more light weight than rbenv

What's the difference between irb and rails console?

TL;DR

The rails console loads the rails env and lets you test ruby code, interact with a DB, use gems, and much more. You need to be in a rails directory to do this
