Every object in ruby is capable of understanding a certain set of messages
Each message that an object understands, corresponds to a method!

Message sending is achieved using the dot operator!

object.message

The message to the right of the dot is sent to the object to the left of the dot

There are other ways to send a message to an object that have slightly more advanced use cases
